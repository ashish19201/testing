git branch     -->>use this to see all the bracnches and an * is shown on the current working branch in the list
git branch  <branchname>   ->> use this command to create a new branch
git switch <branchname>     -> use to switch between branches
git switch -d <branchname>  ->use this to delete a branch
git switch -D <branchname>  ->use this to delete a branch forcefully

git branch -m <branchname>   ->use this to rename a branch and make sure you are on the working branch
